# SZ-Développement | Site Officiel

Bienvenue sur le dépôt du site web de **SZ-Développement**, notre collectif de développeurs étudiants passionnés. Ce site a pour but de centraliser nos réalisations et de présenter notre équipe au monde.

---

## Le Projet

Ce site est conçu pour être la vitrine de notre savoir-faire. Il regroupe :
* **Portfolio :** Nos projets passés et en cours.
* **Membres :** Qui nous sommes et nos spécialités.
* **Contact :** Pour nous proposer des collaborations ou nous rejoindre.
